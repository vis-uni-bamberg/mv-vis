# About

This tutorial has been written by Fabian Beck and Shahid Latif, with technical support by Denis Artjuch. We thank [DataCampus UDE](https://www.uni-due.de/ub/datacampus/index.php), a data literacy initiative by University of Duisburg-Essen, for partly funding the project.

The original tutorial uses R, learnr, and ggplot2. This D3.js version was built with [Observable Framework](https://observablehq.com/framework/) and [D3.js v7](https://d3js.org/).

The world dataset is based on free material from [GAPMINDER.ORG](https://www.gapminder.org), CC-BY LICENSE.

The original tutorial (v1.0.0, May 2022) is archived on Zenodo: [doi:10.5281/zenodo.6523081](https://doi.org/10.5281/zenodo.6523081).
